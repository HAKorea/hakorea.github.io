Media Control 카드

Media Control 카드는 사용하기 쉬운 컨트롤이 있는 인터페이스에 Media Player 엔티티를 표시하는 데 사용됩니다 .

Media Control 카드의 스크린 샷.

Configuration Variables

type

(string)(Required)

media-control

entity

(string)(Required)

미디어 플레이어 entity_id.

예시

type: media-control
entity: media_player.lounge_room
