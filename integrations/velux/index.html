벨룩스(Velux)

Home Assistant의 Velux 연동을 통해 Velux KLF 200 인터페이스에 연결하여 창 및 블라인드와 같은 io-homecontrol 장치를 제어 할 수 있습니다. 이 모듈을 사용하면 KLF 200 내에서 설정된 장면(scene)을 시작할 수 있습니다.

KLF 200 장치에는 2.0.0.0 이상의 펌웨어 버전이 필요합니다. 펌웨어 이미지는 here에서 얻을 수 있으며 KLF 200의 웹 인터페이스를 통해 가져왔을 수 있습니다.

현재 홈어시스턴트에는 다음과 같은 장치 유형이 지원됩니다.

  • Cover
  • Scene

설정

velux 섹션은 configuration.yaml 파일에 있어야하며 필요에 따라 다음 옵션을 포함해야합니다.

# Example configuration.yaml entry
velux:
  host: "192.168.1.23"
  password: "VELUX_PASSWORD"

Configuration Variables

host

(string)(Required)

The IP address or hostname of the KLF 200 to use.

password

(string)(Required)

The password of the KLF 200 interface. Note that this is the same as the WiFi password (in the upper box on the back), not the password for the web login.
