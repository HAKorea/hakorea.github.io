룸바 로봇청소기(irobot Roomba)

roomba 통합구성요소를 통해 iRobot Roomba vacuum을 제어 할 수 있습니다.

이 플랫폼은 테스트를 거쳐 iRobot Roomba 980 및 890 모델과 작동하는 것으로 확인되었지만 690 또는 960과 같은 Wi-Fi 지원 Roomba에서도 잘 작동합니다.

설정

Roomba vacuum을 설치에 추가하려면 configuration.yaml 파일에 다음을 추가하십시오.

# Example configuration.yaml entry
vacuum:
  - platform: roomba
    host: IP_ADDRESS_OR_HOSTNAME
    username: BLID
    password: PASSWORD

Configuration Variables

host

(string)(Required)

The hostname or IP address of the Roomba.

username

(string)(Required)

The username (BLID) for your device.

password

(string)(Required)

The password for your device.

name

(string)(Optional)

The name of the vacuum.

Default value:

Roomba

certificate

(string)(Optional)

Path to your certificate store.

Default value:

/etc/ssl/certs/ca-certificates.crt

continuous

(boolean)(Optional)

Whether to operate in continuous mode.

Default value:

true

delay

(integer)(Optional)

Custom connection delay (in seconds) for periodic mode

Default value:

1

Roomba의 MQTT 서버는 단일 연결만 허용합니다. 연속 모드를 활성화하면 클라우드를 통해 앱이 Roomba에 연결됩니다. More info here

Retrieving your credentials

BLID (사용자 이름)와 비밀번호를 모두 검색하려면 여기 또는 여기를 참조하십시오.
