스마트홈플랫폼(ELV PCA)

pca 스위치 플랫폼을 사용하면 ELV PCA 301 smart switch의 상태를 제어 할 수 있습니다. pca-hex firmware로 플래시 된 [JeeLink]https://www.digitalsmarties.net/products/jeelink)와 같은 868MHz 인터페이스가 필요합니다.

설정

설비에서 PCA 301 스위치 또는 소켓을 사용하려면 configuration.yaml 파일에 다음을 추가하십시오.

# Example configuration.yaml entry
elv:
  device: SERIAL_PORT

이 플랫폼은 범위 내에 있는 모든 PCA 301 스위치를 추가합니다. 총사용 에너지 (KWh)와 현재 전력 (Watt)을 읽을 수 있습니다.

Configuration Variables

device

(string)(Required)

The path to you serial console. Get it via: ls /dev/tty*.

name

(string)(Optional)

Default name for the plugs.
