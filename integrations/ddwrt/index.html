오픈소스라우터(DD-WRT)

이 플랫폼은 연결된 장치를 DD-WRT기반 라우터로 보고 재실 감지 기능을 제공합니다.

설치시 DD-WRT 라우터를 사용하려면 configuration.yaml 파일에 다음을 추가하십시오 :

# Example configuration.yaml entry
device_tracker:
  - platform: ddwrt
    host: ROUTER_IP_ADDRESS
    username: YOUR_ADMIN_USERNAME
    password: YOUR_ADMIN_PASSWORD

Configuration Variables

host

(string)(Required)

The IP address of your router, e.g., 192.168.1.1.

username

(string)(Required)

The username of an user with administrative privileges, usually admin.

password

(string)(Required)

The password for your given admin account.

ssl

(boolean)(Optional)

Whether to connect via HTTPS.

Default value:

false

verify_ssl

(boolean)(Optional)

If SSL/TLS verification for HTTPS resources needs to be turned off (for self-signed certs, etc.)

Default value:

true

wireless_only

(boolean)(Optional)

Whether to only list devices that are connected directly to the router via WiFi or include those connected via Ethernet or other networked access points as well.

Default value:

true

기본적으로 Home Assistant는 5 초마다 DD-WRT에서 연결된 장치에 대한 정보를 가져옵니다. 추적할 사람을 설정하는 방법에 대한 지침은 device tracker integration page를 참조하십시오.
