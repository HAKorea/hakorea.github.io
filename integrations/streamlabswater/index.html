유량관리시스템(StreamLabs)

Streamlabs Water 통합구성요소 플랫폼은 사용 정보를 검색하고 장치의 자리 비움 모드를 관리하기 위해 Streamlabs water monitoring devices와 상호작용하는데 사용됩니다. Streamlabs Water API는 현재 자리 비움 모드와 함께 매일, 매월 및 매년 물사용량을 검색하는데 사용됩니다.

현재 홈어시스턴트에는 다음 장치 유형이 지원됩니다.

  • Binary Sensor
  • Sensor

이 통합구성요소를 사용하려면 Streamlabs API 시작하기 섹션의 지침에 따라 API 키를 요청해야합니다. OAuth 토큰이 아닌 API 키를 요청하십시오.

설정

configuration.yaml 파일에 다음을 추가하십시오 :

# Example configuration.yaml entry
streamlabswater:
  api_key: YOUR_API_KEY

Configuration Variables

api_key

(string)(Required)

Your api_key for the Streamlabs API.

location_id

(string)(Optional)

A specific monitor to use if you have multiple. By default the first found will be used.

set_away_mode 서비스

streamlabswater.set_away_mode 서비스를 사용하여 모드를 home 또는 away로 설정할 수 있습니다. 자리 비움 모드는 설정된 위치에 대해서만 변경됩니다.

Service data attribute Optional Description
away_mode no String, must be away or home.
