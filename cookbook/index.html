쿡북 (홈어시스턴트 설정따라하기)

다음은 홈어시스턴트 설정의 예시들입니다. 대부분의 예시들은 Automation 기능을 사용하여 Automation 관련과 organization의 통합구성요소를 응용하여 설정한 것들입니다.

python_script: 사례의 예시를 살펴보기 위해선 Scripts 색션 포럼을 방문해주세요.

새로운 설정(recipes)은 home-assistant.io 저장소에 추가될 수 있습니다.

최신 유행하는 설정을 찾기 위한 최적의 방법은 home-assistant-config 주제로 GitHub 검색에서 찾는 것입니다.
