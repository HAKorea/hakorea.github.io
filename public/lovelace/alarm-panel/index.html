Alarm Panel Card

The Alarm Panel allows you to Arm and Disarm your Alarm Control Panel Integrations.

Screenshot of the Alarm Panel card. 

type: alarm-panel
entity: alarm_control_panel.alarm

Configuration Variables

type

(string)(Required)

alarm-panel

entity

(string)(Required)

Entity ID of alarm_control_panel domain

name

(string)(Optional)

Overwrites Friendly Name

Default value:

Current State of Alarm Entity

states

(list)(Optional)

Controls which states to have available

Default value:

arm_home, arm_away

arm_home

Arm Home

arm_away

Arm Away

arm_night

Arm Night

arm_custom_bypass

Arm Custom Bypass

theme

(string)(Optional)

Set to any theme within themes.yaml

Examples

Title Example:

- type: alarm-panel
  name: House Alarm
  entity: alarm_control_panel.alarm

Screenshot of the Alarm Panel card.

Define the State List:

type: alarm-panel
name: House Alarm
entity: alarm_control_panel.alarm
states:
  - arm_home
  - arm_away
  - arm_night
  - armed_custom_bypass
